# Scrape Info by Username of a tiktok account

### gets these info :
`unique id`
`nickname`
`user id`
`secUid`
`account creation date`
`nickname last changed date`
`profile last edit date`
`verified status`
`private account status`
`tiktok seller status`
`bio signature`
`bio link url`
`follower count`
`following count`
`hearts received`
`diggs given`
`video count`
`region (country code)`
`full country name with flag emoji`
`language code`
`full language name`
`avatar url`
`linked instagram username`
`linked twitter username`
`linked youtube channel name`
`creator commerce status`
`gender prediction from first name (with confidence percentage)`

Special Feature :
Gets the Real Country of the account even if the user put another country on creation through options. 

made by Axom.
