# Scrape Info by Username of a tiktok account

### gets these info :
- Country/Region
- UniqueId
- Followers
- Following
- Likes/Hearts
- Account Creation Date
- Display Name
- Bio
- avatar URL
- verified status
- private status
- video count
- language code

Special Feature :
Gets the Real Country of the account even if the user put another country on creation through options. 

made by Axom.
